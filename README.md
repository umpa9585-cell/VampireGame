## Vertical Slice (Godot)
A playable prototype lives in `vertical_slice/` and targets Godot 4.x (Xogot compatible). Open `vertical_slice/project.godot` in Godot to run the scene `Main.tscn`.

**Controls**
- WASD: Move
- Shift: Sprint
- Space: Jump
- E: Feed/Interact
- Q: Shadow Dash
- Esc: Toggle mouse capture
